Basics of HTML:

HTML (HyperText Markup Language) is the code that is used to structure a web page and its
content. For example, content could be structured within a set of paragraphs, a list of bulleted
points, or using images and data tables.

Parts of a HTML are:

  • <html>....</html>

  • <head>....</head>

  • <body>....</body>

HTML is all about tags and its attributes.

    Tags and Attributes:


  • Opening and Closing tag

  • meta tag

    1. charset

    2. name

  • title tag

  • strong tag

  • em tag(emphasis)

  • img tag

    1. src(source)

    2. alt

    3. width

    4. height

and many more...
Various tags are there but one important tag is table tag. Let's show this in form of table only so that its better understandable.

HTML table:

Tag Description
table         Defines a table.
tr Defines a row in a table.
th Defines a header cell in a table.
td Defines a cell in table.
